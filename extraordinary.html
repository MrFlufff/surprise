已知集合 A = {x | -2 ≤ x < 3}，B = {x | x ≥ 0}，则 A ∩ B =（　　）

B. [0, 3)
C. [0, 3]
D. (-2, 0]
答题区：
解析：

x ≥ 0 且 -2 ≤ x < 3

∴ A ∩ B = [0, 3)

正确答案：B

2 充分必要条件
"x > 2" 是 "x > 3" 的什么条件？
A. 充分不必要条件
B. 必要不充分条件
C. 充要条件
D. 既不充分也不必要条件
答题区：
解析：

充分条件：若 P 则 Q 成立

必要条件：若 Q 则 P 成立

若 x > 3，则必有 x > 2，但 x > 2 不一定有 x > 3

∴ "x > 2" 是 "x > 3" 的必要不充分条件

正确答案：B

1 证明题
证明集合运算分配律：A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)。
答题区：
证明：

设任意 x ∈ A ∪ (B ∩ C)，则 x ∈ A 或 x ∈ (B ∩ C)

若 x ∈ A，则 x ∈ A ∪ B 且 x ∈ A ∪ C

若 x ∈ B ∩ C，则 x ∈ B 且 x ∈ C，故 x ∈ A ∪ B 且 x ∈ A ∪ C

∴ A ∪ (B ∩ C) ⊆ (A ∪ B) ∩ (A ∪ C)

反之，设任意 x ∈ (A ∪ B) ∩ (A ∪ C)，则 x ∈ A ∪ B 且 x ∈ A ∪ C

若 x ∈ A，则 x ∈ A ∪ (B ∩ C)

若 x ∉ A，则 x ∈ B 且 x ∈ C，故 x ∈ B ∩ C ⊆ A ∪ (B ∩ C)

综上，等式成立

3 集合证明
证明：A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
答题区：
证明：

左 ⊆ 右：若 x ∈ A ∩ (B ∪ C)，则 x∈A 且 x∈B∪C

若 x∈B，则 x∈A∩B ⊆ (A∩B)∪(A∩C)

若 x∈C，则 x∈A∩C ⊆ (A∩B)∪(A∩C)

右 ⊆ 左：若 x ∈ (A∩B)∪(A∩C)，则 x∈A∩B 或 x∈A∩C

若 x∈A∩B，则 x∈A 且 x∈B ⊆ B∪C，故 x∈A∩(B∪C)

同理 x∈A∩C 时也成立

∴ 等式成立

1 解不等式
求不等式 2x² - 5x + 3 ≤ 0 的解集。
答题区：
解析：

解方程 2x² - 5x + 3 = 0：

Δ = 25 - 24 = 1，根 x = (5±1)/4

∴ x₁ = 1.5, x₂ = 1

二次函数开口向上，解为两根之间

∴ 解集为：[1, 1.5]

2 绝对值不等式
解不等式：|2x - 3| ≥ 5
答题区：
解析：

|2x - 3| ≥ 5 等价于：

2x - 3 ≥ 5 或 2x - 3 ≤ -5

⇒ 2x ≥ 8 或 2x ≤ -2

⇒ x ≥ 4 或 x ≤ -1

解集：(-∞, -1] ∪ [4, +∞)

3 分式不等式
解不等式：(x-2)/(x+1) > 0
答题区：
解析：

临界点：x=2 和 x=-1

符号分析：

当 x < -1 时：(-)/(-) = + > 0

当 -1 < x < 2 时：(-)/(+) = - < 0

当 x > 2 时：(+)/(+) = + > 0

∴ 解集：(-∞, -1) ∪ (2, +∞)

4 解方程

6 函数应用
建造一个容积为32m³的长方体水池，底面为正方形。若池壁造价为100元/m²，池底造价为200元/m²，求最低造价。
答题区：

令S'=0：400x³=12800 ⇒ x³=32 ⇒ x=3.17

h=32/(3.17)²≈3.17

S=200×(3.17)² + 12800/3.17 ≈ 4800元

2 证明题
已知 f(x) 是定义在 R 上的奇函数，g(x) = f(x) + 2x，证明 g(x) 既不是奇函数也不是偶函数。
答题区：
证明：

g(-x) = f(-x) + 2(-x) = -f(x) - 2x

而 -g(x) = -f(x) - 2x

g(-x) ≠ g(x) 且 g(-x) ≠ -g(x)

故 g(x) 既不是奇函数也不是偶函数

3 函数方程
已知函数 f(x) 满足 f(x) + 2f(1/x) = 3x (x ≠ 0)，求 f(x) 的表达式。
答题区：
解析：

将 x 替换为 1/x：

f(1/x) + 2f(x) = 3/x

与原方程联立：

设 a = f(x), b = f(1/x)

得：a + 2b = 3x

2a + b = 3/x

解得：a = f(x) = (6/x - 3x)/3 = 2/x - x

最大值 2√2，最小值 -2√2

12 函数性质
证明：函数 f(x) = x³ + x 在 R 上是增函数。
答题区：
证明：

设 x1 < x2

f(x2) - f(x1) = (x2³ + x2) - (x1³ + x1) = (x2³ - x1³) + (x2 - x1)

= (x2-x1)(x2² + x1x2 + x1²) + (x2-x1)

= (x2-x1)(x2² + x1x2 + x1² + 1)

∵ x2 > x1, 且 x2² + x1x2 + x1² = (x1 + x2/2)² + 3x2²/4 > 0

∴ f(x2) > f(x1)

故 f(x) 在 R 上单调递增
